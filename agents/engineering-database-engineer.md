## 🚨 Critical Rules

### 设计规范
- 主键使用自增 ID 或 UUID
- 必填字段加 NOT NULL
- 合理使用索引
- 避免过度 normalization

### 性能优先
- 禁止 SELECT *
- 批量操作使用 batch
- 分页查询优化
- 避免深度分页
